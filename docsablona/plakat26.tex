      {\fontsize{32}{40}\selectfont
      Coach-E je komplexní webová aplikace navržená pro moderní fitness trenéry a jejich klienty. Cílem projektu je zautomatizovat a personalizovat proces tvorby tréninkových a výživových plánů. Systém využívá pokročilé algoritmy pro zpracování vstupních dat od klientů (zranění, alergie, cíle) a generuje optimalizované výstupy. Součástí je také rezervační systém a přímá komunikace, což z Coach-E činí ultimátní nástroj pro správu fitness kariéry.
      }

    \Heading{Motivace a cíle}
    V dnešní uspěchané době je pro trenéry náročné udržet individuální přístup ke každému klientovi. Coach-E řeší tento problém digitální transformací celého workflow. Hlavní cíle byly:
    \begin{itemize}
        \item Automatizace rutinních úkolů trenéra.
        \item Zajištění maximální personalizace pro klienta.
        \item Moderní a intuitivní uživatelské rozhraní.
    \end{itemize}

    \Heading{Technologický stack}
    Projekt byl vyvinut jako robustní full-stack aplikace:
    \begin{itemize}
        \item \textbf{Frontend:} Vite, React, TypeScript – pro rychlé a interaktivní UI.
        \item \textbf{Styling:} Tailwind CSS a Framer Motion pro prémiový vzhled a animace.
        \item \textbf{Backend:} Node.js, Express – spolehlivé API.
        \item \textbf{Databáze:} SQLite – efektivní správa dat.
    \end{itemize}

    \columnbreak

    \Heading{Klíčové vlastnosti}
    \begin{itemize}
        \item \textbf{Dynamické generování plánů:} Plány se automaticky přizpůsobují alergiím, zraněním a preferencím klienta.
        \item \textbf{Multi-role systém:} Dedikované rozhraní pro Administrátora, Trenéra a Klienta.
        \item \textbf{Rezervační a messaging modul:} Integrovaný kalendář pro tréninky a chat.
        \item \textbf{Analytický dashboard:} Sledování pokroku a statistik v reálném čase.
    \end{itemize}

    \Heading{Závěr}
    Coach-E úspěšně propojuje svět IT a fitness. Projekt prokázal, že moderní webové technologie mohou výrazně zvýšit kvalitu a efektivitu ve sportovních službách. V budoucnu je plánováno rozšíření o mobilní aplikaci a integraci nositelné elektroniky.
